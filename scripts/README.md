# 构建脚本

## clean.js

清理打包输出目录 (`release/`)。

### 使用方法

```bash
npm run clean
```

### 功能

- 删除 `release/` 目录及其所有内容
- 如果目录不存在,显示提示信息
- 如果文件被占用,显示警告并退出

### 自动执行

在执行打包命令时会自动运行:
- `npm run package` - 打包前自动清理
- `npm run package:win` - Windows打包前自动清理

### 注意事项

如果清理失败(文件被占用):
1. 关闭所有正在运行的应用实例
2. 手动删除 `release/` 目录
3. 重新运行打包命令
